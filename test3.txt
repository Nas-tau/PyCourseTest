# 5. Дайте определение термину "наследование".
Наследование - это процесс наследования атрибутов и методов одним классом (подкласс) от другого (суперкласс).

# 6. Дайте определение термину "инкапсуляция". Какие модификаторы доступа Вы знаете? 
Инкапсуляция - это органичение доступа к составляющим объект компонентам
Модификаторы доступа: public, private, protected.

# 7. Дайте определение термину "полиморфизм". 
Это разное поведение одного и того же метода в разных классах (к примеру, сложение двух строк и сложение двух чисел).

# 8. Дайте определение термину "абстракция".
В питоне это конструкторы, позволяющие создавать последовательности из других последовательностей.

# 9. Дайте определение термину "композиция". В чем основное отличие "композиции" от "наследования"? 
Композиция - это концепция, моделирующая отношения, позваляет создавать сложные типы, комбинируя объекты других типов.

# 10. Дайте определение термину "итератор"? 
Итератор - это механизм поэлементного обхода данных. Например, 
a = [2,3,4,5]
f = [i**i for i in a]
print(f)

# 11. Что такое "генератор"? В чем его главное отличие от классических итераторов?
Генератор - это функция, которая создает или возвращает последовательность значений. 
Генератор позволяет отложено создавать результат при итерации.

# 12. Что такое "метаклассы"? Что такое "метапрограммирование"?
Метакласс - это классы, экземпляры которого являются классами. 
Метапрограммирование:
 - это вид программирования, 
основанный на особом типе классов - метаклассах, 
иначе говоря:

# 14. Как создать экземпляр класса (объект)?
Экземпляр класса можно создать с помощью метода __init__
Например:
class A(object):
    def __init__(self, name):
        print("Her name is " + name)

class B(A):
    def __init__(self):
        print("She's here")
        super().__init__("Erica")
C = B()
